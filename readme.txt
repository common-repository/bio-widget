== Description ==

Bio Widget is a Example widget that displays a user's bio

== Installation ==

1. Download the zip file and extract the contents.
2. Make sure you're running WordPress version 2.0 or better.
3. Upload the 'bio-widget' folder to your plugins directory (plugins/bio-widget).
4. Activate the plugin through the 'plugins' page in WP.
